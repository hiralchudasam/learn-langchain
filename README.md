## ЁЯМР What is LangChain?

**LangChain** is the most popular open-source framework for building production-ready applications powered by Large Language Models. It gives you composable building blocks to go from a simple LLM call to a full autonomous agent тАФ without reinventing the wheel.

```
Your App тФАтФАтФАтФАтФАтФАтЦ║ LangChain тФАтФАтФАтФАтФАтФАтЦ║ OpenAI / Claude / Gemini / Local LLMs
                     тФВ
              тФМтФАтФАтФАтФАтФАтФАтФ┤тФАтФАтФАтФАтФАтФАтФАтФР
              тФВ   Prompts    тФВ  тЖР Structured, reusable prompt templates
              тФВ   Chains     тФВ  тЖР Composable sequences using LCEL
              тФВ   Memory     тФВ  тЖР Conversation history & summarization
              тФВ   Agents     тФВ  тЖР LLM-driven tool selection & execution
              тФВ   RAG        тФВ  тЖР Retrieval over your private data
              тФВ   LangGraph  тФВ  тЖР Stateful multi-agent orchestration
              тФФтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФШ
```

> **Why LangChain?** Without it, you'd write raw API calls, manually manage conversation history, wire up vector databases from scratch, and build tool-calling logic by hand. LangChain handles all of that тАФ so you focus on what your app does, not how it talks to the LLM.

## тЪб Quickstart

### 1. Install

```bash
pip install langchain langchain-openai python-dotenv
```

### 2. Set your API key

```bash
# .env
OPENAI_API_KEY=your_key_here
```

### 3. Your first chain

```python
from dotenv import load_dotenv
from langchain_openai import ChatOpenAI
from langchain_core.prompts import ChatPromptTemplate
from langchain_core.output_parsers import StrOutputParser

load_dotenv()

chain = (
    ChatPromptTemplate.from_template("Explain {topic} in simple terms.")
    | ChatOpenAI(model="gpt-4o-mini")
    | StrOutputParser()
)

print(chain.invoke({"topic": "RAG"}))
```

That's it тАФ a prompt template, a model, and a parser, connected with `|`. This is **LCEL** (LangChain Expression Language).

## ЁЯзй Core Concepts at a Glance

### LCEL тАФ The Pipe Operator

LangChain Expression Language lets you compose chains elegantly:

```python
# Every component is a Runnable
chain = prompt | llm | parser

# Supports all execution modes
chain.invoke(input)           # single call
chain.batch([i1, i2, i3])    # parallel
chain.stream(input)           # token-by-token streaming
await chain.ainvoke(input)    # async
```

### RAG тАФ Retrieval-Augmented Generation

```python
# Load тЖТ Split тЖТ Embed тЖТ Store тЖТ Retrieve тЖТ Generate
loader   = PyPDFLoader("document.pdf")
splitter = RecursiveCharacterTextSplitter(chunk_size=1000)
store    = Chroma.from_documents(splitter.split_documents(loader.load()), embeddings)

chain = (
    {"context": store.as_retriever(), "question": RunnablePassthrough()}
    | rag_prompt | llm | StrOutputParser()
)
answer = chain.invoke("What does the document say about X?")
```

### LangGraph тАФ Stateful Agents

```python
# Define state + nodes + edges тЖТ compile тЖТ run
graph = StateGraph(AgentState)
graph.add_node("llm", call_llm)
graph.add_node("tools", call_tools)
graph.add_conditional_edges("llm", should_continue, {"tools": "tools", "end": END})
graph.add_edge("tools", "llm")   # loop back тАФ agents can cycle!

app = graph.compile(checkpointer=MemorySaver())
result = app.invoke({"messages": [HumanMessage("Search for LangChain news")]})
```
